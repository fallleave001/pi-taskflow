# pi-taskflow (বাংলা)

> ⚠️ **এই অনুবাদটি পুরোনো।** অনুগ্রহ করে সর্বশেষ তথ্যের জন্য [ইংরেজি README](../../README.md) দেখুন।

pi-taskflow হল [Pi কোডিং এজেন্ট](https://pi.dev)-এর জন্য একটি ডিক্লারেটিভ DAG অর্কেস্ট্রেশন রানটাইম — শূন্য রানটাইম নির্ভরতা, ৫৩৫টি পরীক্ষা, ৯টি ফেজ টাইপ।

```bash
pi install npm:pi-taskflow
```
